Treinar uma rede neural é o processo de **ajustar os pesos internos dos neurônios** para que, a partir dos dados de entrada, a rede consiga produzir as respostas corretas (ou o mais próximo possível delas).  

Imagine que cada neurônio da camada oculta funciona como uma “minicalculadora”. Ele recebe valores normalizados – por exemplo a idade de uma pessoa já transformada em um número entre 0 e 1 – faz algumas contas internas e devolve um novo número. Esse número sai como saída do neurônio e alimenta a próxima camada.  

Se, inicialmente, os pesos dessas “calculadoras” são aleatórios, a rede vai gerar respostas quase ao acaso. **Treinar** significa apresentar à rede muitos exemplos já rotulados (por exemplo, a categoria a que cada pessoa pertence) e, a cada exemplo, comparar a saída que a rede produziu com a saída esperada. A diferença entre elas gera um **erro**, e o algoritmo de treinamento (geralmente o *gradient descent*) ajusta os pesos para reduzir esse erro. Repetindo esse ciclo – alimentar o exemplo, calcular o erro, atualizar os pesos – milhares ou milhões de vezes, a rede “aprende” os padrões escondidos nos dados.

No contexto que você tem:

1. **Normalização**: antes de treinar, as idades são convertidas para o intervalo 0‑1 (por exemplo, 30 → 0.33, 25 → 0, 40 → 1). Isso facilita o aprendizado porque todos os inputs ficam na mesma escala.  
2. **Entrada**: idade, cor, localização, etc., são enviadas para a primeira camada da rede.  
3. **Camada oculta**: cada neurônio combina esses valores de forma diferente, produzindo novas representações.  
4. **Saída**: a rede indica a probabilidade de a pessoa pertencer a cada categoria.  

Ao “treinar a rede para classificar a categoria de uma pessoa”, você está basicamente mostrando à rede vários registros (Erick, Ana, Carlos…) com suas idades normalizadas e a categoria correta. A rede ajusta seus pesos até que, ao receber uma nova pessoa (por exemplo, alguém com idade 28), consiga prever a categoria mais provável.

Em resumo, **treinar** = alimentar a rede com dados rotulados, medir o erro, ajustar os pesos e repetir até que o erro seja suficientemente pequeno. Quando esse processo termina, a rede está pronta para fazer previsões sobre novos dados que ainda não viu.
