Para que o TensorFlow.js possa trabalhar com os seus dados, ele precisa receber **tensores**, ou seja, estruturas que contenham **apenas números**.  
Objetos JavaScript (como `{ nome: "Carlos", idade: 40, cor: "verde", localizacao: "Curitiba" }`) ou valores primitivos (strings, booleans, etc.) não são entendidos diretamente pelo TensorFlow.js. Por isso, antes de criar um tensor, é necessário transformar esses objetos em representações numéricas.

### Passo a passo para converter objetos em tensores

1. **Escolher as propriedades que serão usadas**  
   Decida quais atributos do objeto são relevantes para o modelo. No exemplo do documento usamos:
   - `idade` (valor numérico)
   - `cor` (valor categórico)
   - `localizacao` (poderia ser tratada como categórica ou ignorada)

2. **Converter valores categóricos em números**  
   A forma mais simples é o **one‑hot encoding**: para cada categoria criamos uma coluna que recebe `1` quando a pessoa tem aquela categoria e `0` caso contrário.

   No exemplo temos três cores diferentes (`azul`, `vermelho`, `verde`). A tabela resultante fica:

   | pessoa | azul | vermelho | verde |
   |--------|------|----------|-------|
   | Erick  | 1    | 0        | 0     |
   | Ana    | 0    | 1        | 0     |
   | Carlos | 0    | 0        | 1     |

3. **Montar um array numérico**  
   Junte todas as colunas numéricas (idade + colunas one‑hot) em um único array bidimensional, onde cada linha representa um exemplo:

   ```js
   const dadosNumericos = [
     // [idade, azul, vermelho, verde]
     [30, 1, 0, 0],   // Erick
     [25, 0, 1, 0],   // Ana
     [40, 0, 0, 1]    // Carlos
   ];
   ```

4. **Criar o tensor a partir do array**  
   Use a função `tf.tensor2d` (ou `tf.tensor` quando a forma for diferente) para gerar o tensor:

   ```js
   // Importa o TensorFlow.js
   import * as tf from '@tensorflow/tfjs';

   // Cria um tensor 2‑D (linhas = exemplos, colunas = atributos)
   const tensorPessoas = tf.tensor2d(dadosNumericos);
   console.log(tensorPessoas.shape); // [3, 4]
   ```

   O tensor `tensorPessoas` agora contém apenas números e pode ser passado para camadas, treinamentos ou inferências.

### Resumindo

- **Objetos → números**: extraia os atributos relevantes e converta valores categóricos em vetores one‑hot (ou outra codificação numérica).
- **Array numérico → tensor**: use `tf.tensor`, `tf.tensor2d`, `tf.tensor1d`, etc., dependendo da dimensionalidade dos seus dados.
- **Exemplo completo** (com base no contexto):

```js
import * as tf from '@tensorflow/tfjs';

const pessoas = [
  { nome: "Erick", idade: 30, cor: "azul",   localizacao: "São Paulo" },
  { nome: "Ana",   idade: 25, cor: "vermelho", localizacao: "Rio" },
  { nome: "Carlos",idade: 40, cor: "verde", localizacao: "Curitiba" }
];

// Mapeamento das cores para one‑hot
const cores = ["azul", "vermelho", "verde"];
function corToOneHot(cor) {
  return cores.map(c => (c === cor ? 1 : 0));
}

// Monta o array numérico
const dadosNumericos = pessoas.map(p => [
  p.idade,               // atributo numérico
  ...corToOneHot(p.cor)  // atributos categóricos em one‑hot
]);

// Cria o tensor
const tensorPessoas = tf.tensor2d(dadosNumericos);
tensorPessoas.print(); // exibe o tensor no console
```

Com esse procedimento, você transforma qualquer lista de objetos JavaScript em tensores prontos para serem usados em modelos TensorFlow.js. Boa prática e bons experimentos!
